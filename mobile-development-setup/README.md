# Mobile Development Environment Setup

## Objective

This setup ensures a smooth React Native mobile development experience using the Expo Framework and Expo Go for real-device testing.

## Tools & Requirements

- Node.js (LTS)
- Visual Studio Code
- Windows
- Physical Android device
- Expo Go mobile application

## Expo Go Installation Steps

1. Visited https://expo.dev/go
2. Selected the latest Expo SDK version
3. Installed Expo Go:
   - Android: Google Play Store
4. Opened Expo Go on my mobile device
5. Created an Expo account and logged in successfully

## Why Expo Go

Expo Go allows testing React Native applications directly on a physical device without installing heavy emulators. This makes development faster, cheaper, and more flexible across different device types.

## Challenges Faced

- Initial network mismatch between laptop and mobile device
- Resolved by ensuring both devices were connected to the same Wi-Fi network

## Outcome

Expo Go was successfully installed and connected. The environment is ready for mobile app development and testing.
